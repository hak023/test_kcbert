# KcBERT 욕설/폭언 감지 시스템 - 빠른 배치 처리 가이드

## 🚀 배치 처리 모드

모든 샘플 파일을 한 번에 처리하는 가장 효율적인 방법입니다.

### 실행 방법

#### 방법 1: run.ps1에서 선택

```powershell
.\run.ps1
# → "1. 배치 처리" 선택
```

#### 방법 2: 직접 실행

```powershell
python batch_process.py
```

### 장점

1. **효율적**: 모델을 한 번만 로딩 (첫 파일만 느림)
2. **자동화**: samples 디렉토리의 모든 txt 파일 자동 처리
3. **통계 제공**: 처리 시간, 성공률 등 요약 정보 제공
4. **비교 용이**: 모든 파일의 결과를 한눈에 비교

### 출력 정보

```
📊 전체 처리 결과 요약
======================================================================

📁 처리된 파일: 4개
⚠️  욕설 감지: 1개  
✅ 정상 통화: 3개
⏱️  총 처리 시간: 1.300초
📈 평균 처리 시간: 0.325초/파일

──────────────────────────────────────────────────────────────────────
파일명                       결과              점수         시간(초)     
──────────────────────────────────────────────────────────────────────
abusive_call.txt          ⚠️  욕설 감지       0.9500     0.404     
complaint_call.txt        ✅ 정상            0.3297     0.296     
mixed_call.txt            ✅ 정상            0.3795     0.301     
normal_call.txt           ✅ 정상            0.3641     0.299     
──────────────────────────────────────────────────────────────────────
```

### 처리 시간 해석

- **첫 번째 파일**: 0.4초 (모델 초기화 포함)
- **이후 파일들**: 0.3초 (모델 재사용)
- **평균**: 0.325초/파일

### 커스터마이징

`batch_process.py`를 수정하여:
- 다른 디렉토리 처리
- 필터링 조건 추가
- 출력 형식 변경
- 자동 알림 추가

### 예제: 특정 디렉토리 처리

```python
# batch_process.py 수정
samples_dir = 'my_custom_directory'
txt_files = sorted(glob.glob(os.path.join(samples_dir, '*.txt')))
```

### 활용 사례

1. **일일 배치 처리**: 하루 동안 수집된 통화 기록 일괄 분석
2. **성능 테스트**: 다양한 샘플로 모델 성능 평가
3. **비교 분석**: 여러 파일의 공격성 점수 비교
4. **보고서 생성**: 결과를 자동으로 요약하여 리포트 작성
